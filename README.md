# PLANTORIA

> So vielen Kindern wie moeglich die Moeglichkeit geben, draussen zu lernen.

Bildungsplattform fuer Schulgaerten, die digitales Lernen mit praktischer Gartenarbeit verbindet -- abgestimmt auf den Schweizer Lehrplan 21.
